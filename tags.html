Descrevendo algumas tags HTML* de formatação de texto

Tag <em>

Essa tag é utilizada para dar enfase ao texto em uma pagina html.O seu comportamento padrão exibe o texto em italico.

Tag <strong>

Esta tag é utilizada para dar importância ao texto.O seu comportamento padrão exibe o texto em negrito

Tag <sup> e <sub>

Estas tags são utilizadas para definir textosobrescritoe subscrito,respectivamente.São muito utilizadas em formulas quimicas (H2SO4) e expressões matematicas com potência(23=8),por exemplo.

Tag <ins> e <del>

Estas tags são utilizadas para definir inserções de texto (com a tag<ins>) e remoções de texto (com a tag <del>).Inserções e remoções são comuns em artigos de blogs, quando o autor quer deixar o usuario ciente de gouve uma atualizarão atualização no text, seja ela uma correção, inserção ou inclusão de palavra(s).O comportamento padrão deixa sublinhado para inserções e tachado para exclusões.

Tag <mark>

Esta tag é utilizada para marcar trechos do texto. É possivel marcar tanto palavrasespecíficas quanto um conjunto de palavras.. O comportamento padrão deixa o texto na cor preta e com o fundo amarelo.

*HTML: Hyper Text Markup Language
